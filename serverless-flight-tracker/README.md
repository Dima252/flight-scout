# serverless-flight-tracker

The Storefront: Explains your project to recruiters on LinkedIn.

## Setup

This tracker uses the free [Tequila API by Kiwi.com](https://tequila.kiwi.com/). To run this project:
1. Create a free account at Tequila by Kiwi.com
2. Go to your GitHub Repository Settings > Secrets and Variables > Actions
3. Add a new secret named `TEQUILA_API_KEY` and paste your key.
4. Set up your Gmail App Password and add `EMAIL_SENDER`, `GMAIL_APP_PASSWORD`, and `EMAIL_RECEIVER` as secrets.
